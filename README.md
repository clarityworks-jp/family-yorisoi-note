# 家族を想う よりそいノート

親御さんや祖父母など、大切な家族のことが少し気になったときに、話すこと・そっと気づくこと・家族で共有することをやさしく整理する無料のPWAです。

## コンセプト

大切な家族をひとりの人として尊重し、本人の気持ちとペースを大切にしながら、会話を少しだけ始めやすくする準備ノートです。すべてを一度に話す必要はありません。

## 使い方

1. `index.html` をブラウザで開きます。PWAとしての動きを見る場合は、ローカルサーバーまたはGitHub Pagesから開いてください。
2. 想っている家族、距離感、気になっていること、会話の雰囲気、今回の目的を選びます。
3. 「会話のしおりを作る」を押します。
4. 結果の中から、今回使えそうな言葉を「今日はこれ」としてひとつ選べます。
5. 必要な部分だけコピーするか、結果全体をコピー・印刷して使います。

結果カードは画面上で開閉できます。印刷するときは、開閉状態にかかわらずすべてのカードが表示されます。「今日使うひとつ」は、選ばれている場合だけ印刷されます。

## 技術構成

- HTML / CSS / JavaScriptのみ
- 外部API、外部ライブラリ、外部フォント不使用
- `manifest.json` と `service-worker.js` によるPWA対応
- アプリ本体をキャッシュし、基本画面をオフラインでも表示
- スマホファーストのレスポンシブUI
- 印刷用CSS

## GitHub Pagesで公開する方法

1. この `family-yorisoi-note` フォルダーの内容をGitHubリポジトリへ追加します。
2. GitHubのリポジトリ画面で **Settings → Pages** を開きます。
3. **Build and deployment** のSourceを **Deploy from a branch** にします。
4. 公開するブランチと、ファイルを置いたフォルダーを選んで保存します。
5. 表示された公開URLへアクセスします。

サブディレクトリで公開できるよう、ファイル参照には相対パスを使用しています。

## 入力内容とプライバシー

入力内容は外部へ送信しません。入力内容の `localStorage` への自動保存も行いません。入力内容は現在の画面内だけで使われ、ページを閉じるか再読み込みすると消えます。

文字サイズの設定だけは、次回も同じ大きさで表示するため、この端末の `localStorage` に保存します。

## 広告・関連リンク

関連する備えのカードは `app.js` の `relatedCardCatalog` 配列にまとめています。各カードの `url` に案内ページや広告・アフィリエイトリンクを設定すると、リンクとして表示されます。

`url` が空のカードはクリックできるリンクにならず、「関連ページは準備中です」と表示されます。案内先が整ったものから、無理なく差し替えられる構成です。

広告などの外部リンク先はService Workerのキャッシュ対象に含めていません。
